Windows DIR Commands

Click here to show/hide DIR commands

  • dir - Lists files and directories in the current directory.
  • dir /a - Displays all files including hidden and system files.
  • dir /a:h - Displays hidden files.
  • dir /a:s - Displays system files.
  • dir /o - Sorts the directory list:
    • /o:n - by name
    • /o:e - by extension
    • /o:d - by date/time
    • /o:s - by size
  • dir /s - Lists all files in the current directory and its subdirectories.
  • dir /b - Bare format (file names only).
  • dir /p - Pauses after each screen of output.
  • dir /w - Wide list format (only file names).
  • dir /t:c - Uses creation time for sorting.
  • dir /t:w - Uses last write time for sorting.
  • dir /t:a - Uses last access time for sorting.
  • dir /q - Displays file owner.
  • dir /x - Shows short (8.3) file names.
  • dir /4 - Displays four-digit year format.
  • dir /a /s /p - Lists all files including hidden ones, in all subdirectories, pausing each screen.
  • dir /o:n /s - Lists sorted by name, including all subdirectories.
